Blog 2 - September 9 2021

This week, a useful element we learned about was style sheets. We used internal stylesheets in particular, which could be set
within the html page. These stylesheets are useful when you would like to use a repeated element within the same page. For example
the following code is actually its own style sheet -

{ color: blue; background-color: lightgrey; border-style:double; border-color: black; }

These layouts were suprisingly similar, including many of the same features and concepts, to Java Swing, which I have experience using in CS 159. The default layout, flow, which moves elements from left to right within a container, was intuitive to figure out. This example uses grid-template-columns: 20px 20px 20px; , which, in one line, sets the number of columns and the size of each grid. Each of these layouts can use the justify-content property to easily set the positioning of items within the container, such as space-between which will create space between items, and which will center items within the container. So far, most of these layouts and their associated properties are easy to work with, although parent and child relationships are a bit tricky but can be figured out.
